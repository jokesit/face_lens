# Checklist ก่อนนำ FaceLens ไปทดลองใช้ในร้านยา

## ก่อนแจกไฟล์

- [ ] Build ด้วย `scripts\build_windows.ps1`
- [ ] สร้าง release ZIP ด้วย `scripts\create_release_zip.ps1`
- [ ] ตรวจว่า ZIP ไม่มีข้อมูลลูกค้าจริงหรือฐานข้อมูลทดสอบที่ไม่ควรส่งต่อ
- [ ] เปิด `FaceLens.exe` จากโฟลเดอร์ release แล้วทดสอบกล้อง
- [ ] กด `ตรวจระบบ` แล้วไม่มี error สำคัญ

## ตอนติดตั้งที่ร้าน

- [ ] แตก ZIP ทั้งโฟลเดอร์ ห้ามย้ายเฉพาะ `FaceLens.exe`
- [ ] เลือกกล้องที่ใช้จริง
- [ ] ตั้งโหมดความเร็วเป็น `สมดุล (แนะนำ)`
- [ ] ตั้งความมั่นใจเป็น `ปลอดภัยสูง` ในช่วงเริ่มต้น
- [ ] เพิ่มลูกค้าทดสอบ 1–2 คน
- [ ] ทดสอบ backup และ restore อย่างน้อย 1 ครั้ง

## หลังทดลองใช้

- [ ] ถามพนักงานว่าหน้าจอใช้ง่ายหรือไม่
- [ ] ตรวจว่ามีการทักชื่อผิดหรือไม่
- [ ] ตรวจรายชื่อซ้ำเป็นระยะ
- [ ] Backup ฐานข้อมูลก่อนอัปเดตเวอร์ชันใหม่
