# คู่มือใช้งาน FaceLens สำหรับร้านยา

## 1. การวางกล้อง

- วางกล้องให้เห็นใบหน้าลูกค้าชัดเจนในระดับสายตา
- หลีกเลี่ยงแสงย้อนจากประตูหรือหน้าต่าง
- ถ้าหน้าเบลอ ให้เพิ่มแสงหรือขยับกล้องให้ใกล้ขึ้น

## 2. การเพิ่มลูกค้า

1. กด `เพิ่ม / อัปเดตข้อมูลลูกค้า`
2. กรอกชื่อลูกค้าให้สั้นและชัดเจน เช่น `คุณสมชาย`
3. ให้มีใบหน้าเพียง 1 คนในกล้อง
4. รอให้ระบบเก็บภาพครบตามจำนวนที่กำหนด
5. กดบันทึก

คำแนะนำ: สำหรับฐานข้อมูลหลักพันคน ควรเก็บใบหน้าคุณภาพดี 5–8 รายการต่อคน ไม่จำเป็นต้องเก็บเยอะเกินไป

## 3. การตั้งค่าความเร็ว

- `เบาและเร็ว`: เหมาะกับเครื่องสเปกต่ำ ลดการเรียก AI
- `สมดุล (แนะนำ)`: เหมาะกับร้านทั่วไป
- `แม่นยำขึ้น`: เหมาะกับเครื่องแรงหรือช่วงทดสอบ

## 4. การตั้งค่าความมั่นใจ

- `ปลอดภัยสูง`: ลดโอกาสทักผิดชื่อ เหมาะกับช่วงเริ่มใช้งานจริง
- `สมดุล (แนะนำ)`: ใช้ทั่วไป
- `ทักง่ายขึ้น`: เหมาะเมื่อมีข้อมูลลูกค้าเยอะและทดสอบแล้วว่าแม่น

## 5. การจัดการลูกค้า

หน้า `จัดการข้อมูลลูกค้า` ใช้สำหรับ:

- ดูรายชื่อลูกค้า
- แก้ชื่อ/หมายเหตุ
- พักการจดจำชั่วคราว
- ลบออกจากการจดจำ
- ลบถาวรตามคำขอลูกค้า
- ตรวจรายชื่อที่อาจซ้ำกัน

## 6. Backup / Restore

- ใช้ `สำรองฐานข้อมูล` อย่างน้อยสัปดาห์ละครั้ง
- เก็บไฟล์ backup ไว้นอกเครื่อง เช่น external drive หรือ cloud storage ของร้าน
- ใช้ `กู้คืนฐานข้อมูล` เมื่อต้องย้ายเครื่องหรือกู้ข้อมูล

## 7. Maintenance

ใช้ปุ่ม `ล้างประวัติเก่า` และ `ปรับฐานข้อมูลให้เร็วขึ้น` เป็นระยะ โดยเฉพาะเมื่อใช้งานมานานหรือมีการลบข้อมูลจำนวนมาก
