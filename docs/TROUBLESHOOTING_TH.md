# คู่มือแก้ปัญหา FaceLens

## โปรแกรมเปิดไม่ได้

1. เปิดโฟลเดอร์ `logs`
2. ส่งไฟล์เหล่านี้ให้ผู้ดูแลระบบ:
   - `facelens_runtime.log`
   - `facelens_startup.log`
   - `facelens_crash.log` ถ้ามี
3. ลองรัน `FaceLensDebug.exe` ถ้ามีชุด debug build

## กล้องไม่ขึ้น

- ลองเปลี่ยนช่อง `กล้อง` เป็น 0, 1, 2 หรือ 3
- ปิดโปรแกรมอื่นที่ใช้กล้อง เช่น Zoom, LINE, OBS
- ถอดแล้วเสียบกล้อง USB ใหม่
- กด `ตรวจระบบ`

## โปรแกรมจำชื่อผิด

- เปลี่ยน `ความมั่นใจ` เป็น `ปลอดภัยสูง`
- เพิ่มรูปใบหน้าที่ชัดขึ้นให้ลูกค้าคนนั้น
- ตรวจว่าลูกค้าคนเดียวกันถูกบันทึกซ้ำหลายชื่อหรือไม่
- ใช้ปุ่ม `ตรวจรายชื่อซ้ำ`

## โปรแกรมช้า / เครื่องร้อน

- เปลี่ยน `โหมดความเร็ว` เป็น `เบาและเร็ว`
- ลดจำนวนโปรแกรมที่เปิดพร้อมกัน
- ตรวจว่าฐานข้อมูลมีประวัติเยอะเกินไปหรือไม่ แล้วใช้ `ล้างประวัติเก่า`

## ข้อมูลลูกค้าหาย

- ตรวจว่าเปิดโปรแกรมจากโฟลเดอร์เดิมหรือไม่
- ใช้ปุ่ม `กู้คืนฐานข้อมูล` แล้วเลือกไฟล์ backup ล่าสุด
- อย่าย้ายเฉพาะ `FaceLens.exe` ออกจากโฟลเดอร์ ต้องย้ายทั้งโฟลเดอร์ `FaceLens`
