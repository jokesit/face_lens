# FaceLens 0.16

FaceLens คือโปรแกรมช่วยร้านขายยาจำชื่อลูกค้า โดยใช้กล้องและ AI จดจำใบหน้าแบบ standalone บนเครื่องคอมพิวเตอร์ของร้าน

## วิธีติดตั้งที่แนะนำ

สำหรับร้านขายยาจริง แนะนำใช้ไฟล์ติดตั้ง:

```text
FaceLens_0.16_Setup.exe
```

ตัวติดตั้งจะทำให้อัตโนมัติ:

```text
ติดตั้งโปรแกรมที่: C:\Program Files\FaceLens
สร้างไอคอนบน Desktop: FaceLens
สร้างเมนูใน Start Menu: FaceLens
เก็บฐานข้อมูลร้านที่: C:\ProgramData\FaceLens
```

หลังติดตั้ง ให้เปิดจากไอคอน **FaceLens** บน Desktop ได้เลย

## ข้อมูลร้านถูกเก็บไว้ที่ไหน

ข้อมูลลูกค้า ฐานข้อมูล ใบหน้า backup และ log จะไม่เก็บในโฟลเดอร์โปรแกรม แต่เก็บที่:

```text
C:\ProgramData\FaceLens
```

โครงสร้างสำคัญ:

```text
data\facelens.db       ฐานข้อมูลหลัก
backups\               ไฟล์สำรองฐานข้อมูล
logs\                  ไฟล์ log สำหรับช่วยแก้ปัญหา
.deepface\weights\     ไฟล์โมเดล AI ของ DeepFace
```

## การใช้งานเริ่มต้น

1. เปิด FaceLens จาก Desktop
2. เลือกกล้องให้ถูกต้อง หากกล้องไม่ขึ้น
3. ใช้โหมดความเร็ว: **สมดุล (แนะนำ)**
4. ใช้ความมั่นใจ: **ปลอดภัยสูง** หรือ **สมดุล (แนะนำ)**
5. เพิ่มลูกค้าทดสอบ 1-2 คนก่อนใช้งานจริง
6. กดสำรองฐานข้อมูลเป็นประจำ

## ข้อควรระวัง

ใบหน้าและ embedding เป็นข้อมูลอ่อนไหว ร้านควรแจ้งลูกค้าก่อนบันทึกข้อมูล และลบข้อมูลให้เมื่อลูกค้าร้องขอ

## การขอความช่วยเหลือ

หากโปรแกรมเปิดไม่ได้หรือทำงานผิดปกติ ให้ส่งโฟลเดอร์นี้ให้ผู้ดูแล:

```text
C:\ProgramData\FaceLens\logs
```
